# 🆔 Easy ID

> A modern, fast and open-source Telegram bot to retrieve Telegram IDs and useful account information.

![Version](https://img.shields.io/badge/version-v0.0.1--alpha-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Telegram](https://img.shields.io/badge/Telegram-Bot-2CA5E0)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6)
![grammY](https://img.shields.io/badge/Framework-grammY-ff9800)

---

## ✨ Features

- 👤 Get your Telegram User ID
- 👥 Get Group & Supergroup ID
- 📢 Get Channel ID
- 💬 Get Message ID
- 🖼 Send profile photo (if available)
- ⭐ Detect Telegram Premium
- 🤖 Detect Bot accounts
- 🌍 Show user's language
- 🔗 Mention links
- 🛰 Deep Scan
- 🔍 Whois lookup
- 📸 Profile photo counter
- ⚡ Fast & lightweight
- 🌐 Open Source

---

## 🚀 Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/help` | Show help |
| `/id` | Show your ID or replied user's ID |
| `/whois` | Detailed information about a user |
| `/deepscan` | Advanced scan report |
| `/about` | About Easy ID |

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/F2Codes/EasyID-Project.git
```

Go to the project:

```bash
cd EasyID-Project
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
BOT_TOKEN=YOUR_BOT_TOKEN
```

Start the bot:

```bash
npm run dev
```
